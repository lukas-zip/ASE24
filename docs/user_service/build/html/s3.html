S3 Bucket¶

s3.bucket_exists(bucket_name)¶

Checks if an S3 bucket exists by attempting to retrieve its metadata.

Parameters:

bucket_name – The name of the bucket to check.

Returns:

True if the bucket exists, False if the bucket does not exist or an error occurs.

Raises:

ClientError – If an unexpected error occurs during the check (other than 404 not found).

s3.create_bucket(bucket_name)¶

Creates an S3 bucket with the specified name using the Boto3 S3 client.

Parameters:

bucket_name – The name of the bucket to create.

Returns:

None, prints success or error message.

Raises:

ClientError – If there’s an error during the bucket creation process.

s3.delete_object(object_key, bucket_name)¶

Deletes an object from an S3 bucket.

Parameters:

  • object_key – The key of the object to delete.

  • bucket_name – The name of the bucket from which the object will be deleted.

Returns:

True if the object is successfully deleted, False otherwise.

Raises:

ClientError – If there’s an error during the deletion process.

s3.retrieve_image(object_key, bucket_name)¶

Retrieves image data from an S3 bucket at the specified object key.

Parameters:

  • object_key – The key of the object to retrieve.

  • bucket_name – The name of the bucket from which the object is retrieved.

Returns:

The image data as a bytes object if successful, None otherwise.

Raises:

ClientError – If there’s an error retrieving the object.

s3.upload_fileobj(image_file, bucket_name, object_key)¶

Uploads a file object to an S3 bucket at the specified object key.

Parameters:

  • image_file – The file-like object to upload.

  • bucket_name – The name of the S3 bucket where the file will be uploaded.

  • object_key – The key under which the file object will be stored.

Returns:

True if the upload is successful, False otherwise.

Raises:

ClientError – If there’s an error during the file upload process.
