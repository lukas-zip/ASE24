dynamodb.add_shop(shop_name, email, password, address, phone, description, profile_picture, shop_pictures, dummyid=None)¶

Adds a new shop to the DynamoDB ‘UserManagement’ table with provided details.

Parameters:

  • shop_name – Name of the shop.

  • email – Shop owner’s email address.

  • password – Shop owner’s password (will be hashed before storage).

  • address – Shop’s address (optional).

  • phone – Shop’s phone number (optional).

  • description – Description of the shop (optional).

  • profile_picture – URL to the shop’s profile picture (optional).

  • shop_pictures – List of URLs to pictures of the shop (optional).

  • dummyid – Optional UUID to use instead of generating a new one.

Returns:

JSON dictionary containing shop data if successful, None if shop exists.

Raises:

ClientError – If there is an error during the shop addition process.
